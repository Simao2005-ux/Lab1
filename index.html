Programação em html

Na Escola do Comércio de Lisboa, na disciplina de Tecnologias 1 no 1ATIS iremos aprender Programas WEB 😅 e abordaremos os princípios básicos associados ao desenvolvimento WEB, apenas do lado Front-End, lado visual. Também sabemos que existe um lado Back-End, que é o lado do servidor, mas não iremos abordar esse lado.

Aprender-se-ão as segundas tecnologias 🌐:

  • HTML
  • CSS
  • JavaScript

De forma a consolidar todos os conhecimentos, serão feitos alguns laboratórios.
